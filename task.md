构建一个可复现的DVC pipeline，用来测试素自动语音识别（ASR）系统对噪声的鲁棒性 。

- [x] 使用使用 espeak-ng 工具，将原始的参考文本转换为音素序列 。

- [x] 使用add_noise 和 add_noise_to_file 函数，为原始音频生成多个不同信噪比（SNR）的加噪版本 。

`add_noise` 读取波形并估算其平均功率。根据SNR生成功率完全匹配的白高斯噪声 。把这段计算好的噪声直接叠加到原始波形上，并输出最终的带噪信号 。

`add_noise_to_file` 读取原始音频文件和把处理好的音频保存到硬盘上。

- [x] 利用 HuggingFace 库，加载预训练模型 facebook/wav2vec2-lv-60-espeak-cv-ft 进行推理 。这个模型会接收原始音频波形，并输出预测的音素标记（tokens） 。你需要将这一步作为一个独立的流水线阶段，读取上一步骤生成的清单文件，并输出包含预测结果的新清单文件 。

- [x] 使用音素错误率（PER）来评估模型在不同噪声下的表现 。

- [x] 结果可视化与多语言扩展。
